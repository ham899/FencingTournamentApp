# Core Features (MVP)

## Tournament Workflow

### 1. Tournament Setup
- Create tournament
- Add participants manually or from saved list
- Select format:
  - Standard (5 / 15)
  - Kids (5 / 10)
  - Custom

### 2. Poule Generation
- Automatically generate poules
- Display match schedule

### 3. Match Execution
- Enter results in any order
- Edit results
- Real-time updates:
  - Victories (V)
  - Matches fenced
  - TS / TR
  - Indicator
  - Rankings

### Ranking Rules
1. Victory Ratio
2. Indicator
3. Touches Scored
4. Random tie-break

### 4. Direct Elimination (DE)
- Generate bracket from rankings
- Auto-seeding

### 5. DE Matches
- Enter results
- Auto-advance winners
- Update bracket live

### 6. Completion
- Display final rankings
- Save or discard tournament
