# Fencing Tournament App

A mobile-friendly web application for running small, in-house fencing tournaments, designed to replace paper-based workflows during live training sessions.

## Overview

This project aims to replace manual tournament processes with a simple digital system that allows coaches to:

- Generate poules
- Record match results
- Automatically compute rankings
- Run direct elimination (DE) brackets
- View final results in real time

The application is designed for quick setup and use during live training sessions.

## Features (MVP)

- Tournament creation and setup
- Automatic poule generation
- Flexible match result entry and editing
- Real-time ranking computation
- Direct elimination bracket generation
- Final standings display
- Optional saving of tournament data

## Tech Stack

- **Backend:** Python
- **API:** FastAPI
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite (development), cloud-hosted database (planned)
- **Testing:** pytest
